~ 何かあったときのメモ ~
・プレーンテキストはPlantUMLに(じゃないと、UML認識されない)
・ターミナルはbashにしよう


~ 覚えておきたいショートカット ~
・Ctrl + B          => エクスプローラーを閉じたり開いたり
・Ctrl + Shift + T  => 最後に閉じたエディタを再表示
・Ctrl + D          => 複製（"D"はduplicateの意味！！）
・Ctrl + W          => ファイルを閉じる
・Alt + Shift + ↓   => 行を増やす
・Alt + Shift + ↑   => 選択された行を上の行と入れ替える
・Ctrl + /          => コメントアウト
